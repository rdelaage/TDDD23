# Fall To The Top

A game by Romain de Laage for TDDD23 course.

## Export

Exports are in the `export` folder. Exports are available for:
* Linux
* HTML5
* MacOS
* Windows

You can play the game on https://tddd23.patu.re/FallToTheTop.html
